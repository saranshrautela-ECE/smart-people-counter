# Smart People Counter and Display System

Arduino-based smart people counter using IR sensors and P10 LED display for real-time room occupancy monitoring.

---

## Features
- Real-time people counting
- Entry and exit detection
- P10 LED display output
- Room occupancy monitoring
- Buzzer alert system

---

## Components Used
- Arduino Uno
- IR Sensors
- P10 LED Display
- Buzzer
- Jumper Wires
- Power Supply

---

## Working Principle
Two IR sensors are used to detect entry and exit movement.  
The Arduino processes sensor signals and updates the people count.  
The count is displayed on the P10 LED display in real time.
